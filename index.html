E-Commerce Website Backend

In the exciting journey of my coding education at the Zuitt Coding Bootcamp, one of the standout accomplishments is the completion of my backend capstone project. This project is a testament to my growing skills in web development, where I utilized the power of JavaScript and harnessed the robustness of MongoDB as its database.

This capstone is designed to cater to the complex needs of an Ecommerce website. It's not just another project; it's a showcase of my ability to build a backend that supports a dynamic online store, complete with products, orders, and user management.

One key feature that sets this project apart is the dual account system. It seamlessly integrates both admin and non-admin accounts, giving the platform a versatile edge. This means users can interact with the website from different angles, whether they're regular customers or administrators managing the website's backend operations.
